◆ PDALVersionInteger()

PDALC_API int PDALVersionInteger ( )

Returns an integer representation of the PDAL version.

The returned value is the sum of ther major version number multiplied by 10,000, the minor version number multiplied by 100, and the patch version number. For example, this method returns 10702 for version 1.7.2.
